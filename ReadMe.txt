---------------------------------------------------------------------
--------------------------USER GUIDE---------------------------------
---------------------------------------------------------------------
Tools:
	- Python v3.8
	- Flask Framework
	- Experta Library
---------------------------------------------------------------------
Steps:

1)  Install python
2)  Create a project directory
3)  Cd to the directory 
4)  Create virtual environment using: virtualenv envName
5)  Activate the virtual environment using: envName\Scripts\activate
6)  Install Python packages using: pip install -r requirements.txt
7)  Install Flask framework using: pip install flask
8)  Install Experta library using: pip install experta
9)  Run the project using: flask run
10) Follow the URL: http://localhost:5000/
11) And let the show begin ^_^
